# Estrategia de Diseño Web y Funnels Mconector (SKILL)

Este documento recopila la metodología de alta conversión utilizada para el proyecto Mconector. Úsalo como guía para replicar el éxito en futuras páginas.

## 1. Psicología del Copy: Framework PAS
La clave es no vender características técnicas primero, sino atacar problemas reales:
- **P (Problem):** "Estás perdiendo llamadas y dinero."
- **A (Agitate):** "Cada llamada perdida es un contrato de $20k regalado a la competencia."
- **S (Solution):** "Mconector organiza todo por ti de forma simple."

## 2. Arquitectura "Hub & Spoke"
- **Página de Inicio (Hub):** Vitrina general de soluciones.
- **Landings de Producto (Spokes):** Una página larga por cada nivel de servicio (`base-digital`, `sistema-comercial`, `asistencia-inteligente`). Esto permite enviar tráfico de Ads específico a cada dolor.

## 3. Estrategia de Conversión (CRO) Superior
- **CTAs de Baja Fricción:** Cambiamos de "Agendar Cita" a "Solicitar Diagnóstico". Es mucho más fácil que alguien deje su teléfono a que se comprometa con una fecha en un calendario.
- **Value Stacking:** Incluir listas masivas de "Todo lo que recibes" (Checklists técnicas) para que el precio de +$400 parezca una ganga.
- **Ocultar Precios:** En las páginas de venta directa, ocultamos el precio para priorizar que el cliente pida la llamada. El valor se vende en vivo.

## 4. Estética Premium
- **Fondo:** `#0F172A` (Azul Noche).
- **Tipografía:** `Manrope` (Titulares) e `Inter` (Cuerpo).
- **Interactividad:** Animaciones 3D basadas en scroll (Canvas Image Sequence) para generar "Engagement".
- **Glassmorphism:** Efectos de desenfoque y capas para dar profundidad tecnológica.

## 5. Mobile First
- El 90% de los anuncios se ven en móvil. Los botones deben ser enormes y el formulario debe tener máximo 3 campos (Nombre, Tel, Negocio).
